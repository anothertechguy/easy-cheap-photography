# Agent Instructions

Easy Cheap Photography — Astro static site deployed to Cloudflare Pages.

## Stack
- **Framework**: Astro (static output)
- **Styling**: Tailwind CSS v4 (via `@tailwindcss/vite` Vite plugin)
- **React**: Only for `src/components/Booking.tsx` (form state island)
- **Images**: Astro `Image` component auto-optimizes to WebP + srcset
- **Sitemap**: Auto-generated by `@astrojs/sitemap` on build
- **Package manager**: npm

## Key files
- `src/pages/index.astro` — full page, all SEO meta
- `src/components/` — Astro components + `Booking.tsx` React island
- `src/styles/global.css` — Tailwind v4 theme tokens + animations
- `src/assets/` — source images (hero.jpg, g1-g8.jpg)
- `astro.config.mjs` — Astro config
- `public/` — favicon.svg, robots.txt

## Dev commands
```bash
npm run dev      # dev server at localhost:4321
npm run build    # static build → dist/
npm run preview  # preview build
```

## Cloudflare Pages settings
- Framework preset: Astro
- Build command: `npm run build`
- Output directory: `dist`
